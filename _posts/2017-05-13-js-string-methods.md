Js string most used methods 
------------------------

### js string methods 

##### js slice
```
> var a = '12345'
> var b = a.slice(1, 3)
> b
< '23'
```
##### js repeat
```
> var a = '0'
> var b = a.repeat(3)
> b
< '000'
```
##### js includes
```
> var a = '2'
> var b = '67243'
> var c = b.includes(a)
> c
< true
```
##### js indexOf
```
> var a = '2'
> var b = '64289'
> var c = b.indexOf(a)
> c
< 2
```
